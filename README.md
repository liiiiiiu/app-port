# 应用端口

记录前端开发时各项目在不同环境下使用的端口号，防止端口冲突。

## 使用

克隆代码到本地后，打开谷歌浏览器，点击 `更多工具 -> 扩展程序 -> 加载已解压的扩展程序`；

在根目录下新增 data.json，并按以下规则输入数据：

```json
{
  "ports": [
    {
      // 应用名称
      "title": "应用A",
      // 应用描述，可不填
      "sub_title": "描述A",
      // 开发环境使用的端口号，可不填
      "dev_port": 3000,
      // 测试环境使用的端口号，可不填
      "test_port": 6000,
      // 生产环境使用的端口号，可不填
      "prd_port": 8000
    },
    {
      "title": "应用B",
      "sub_title": "",
      "dev_port": 3001,
      "test_port": null,
      "prd_port": null
    },
    // ...
  ]
}
```
